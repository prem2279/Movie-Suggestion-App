# 🎬 Movie Suggestion App (SwiftUI)

A simple and elegant **Movie Suggestion App** built using **SwiftUI** that helps users discover movies based on genres and curated lists. The app focuses on clean UI, smooth interactions, and a modular SwiftUI structure.

---

## ✨ Features

- 🎥 Browse movies by genre  
- 🔍 View movie details (title, genre, description, rating)  
- ❤️ Favorite movies (optional feature)  
- 🎨 Clean and modern SwiftUI UI  
- ⚡ Fast and responsive interface  
- 📱 Native iOS experience  

---

## 🧱 Tech Stack

- SwiftUI  
- Swift  
- MVVM Architecture  
- Combine (optional)  
- Xcode
